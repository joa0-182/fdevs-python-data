# FDEVS - Trilha de Análise de Dados com Python

Bem-vindo ao repositório oficial da trilha de **Análise de Dados com Python** do programa **FDEVS**.

Este material foi desenvolvido para apoiar alunos iniciantes que desejam dar os primeiros passos na área de dados, com foco em aplicações práticas e conexão com o mercado de trabalho.

---

## Sobre o Programa

O **FDEVS** é um programa de capacitação promovido pela UPPERCASE Consultoria, existente desde 2010.

O objetivo é preparar jovens talentos para o mercado de tecnologia, combinando:
- Fundamentos técnicos
- Aplicação prática
- Visão de negócio

---

## Objetivo da Trilha

Nesta trilha, você irá aprender:

- Fundamentos de análise de dados
- Manipulação de dados com Python
- Uso de bibliotecas como `pandas`, `numpy` e `matplotlib`
- Estruturação de pipelines simples
- Boas práticas de organização de código
- Interpretação de dados com visão de negócio

O foco não é apenas **codar**, mas **entender o que está acontecendo** desenvolver o pensamento analítico.

---

## Abordagem Didática

Este repositório segue alguns princípios importantes:

- Simplicidade acima de complexidade
- Código legível e explicativo
- Estrutura fácil de entender
- Pensado para iniciantes (zero ou pouca experiência)

Se algo parecer confuso, provavelmente pode ser simplificado, e esse é o objetivo aqui.
